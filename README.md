### What I do

I specialize in **AI Agent systems** — the kind that actually work in production, not just in demos.

That means tool-calling pipelines, RAG architectures, LLM orchestration, and making sure none of it falls apart when real users touch it. I write **TypeScript** when things need to be fast and maintainable. I write **Python** when things need to be smart.

Most of my work sits at the intersection of backend engineering and applied AI — building systems that automate decisions, not just tasks.
